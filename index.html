가족농장 컨테이너 셀프 리모델링 파이프라인 v2

법적·행정 검토는 완료된 것으로 가정하고, 방/부엌 우선, 화장실은 2차 공정으로 분리한 실행용 문서입니다. 이번 버전은 외피 방수 → 창호 → 구조·설비 → 방/부엌 마감 → 기기 설치 → 화장실 순서로 재정렬하고, 현장 조건에 맞는 자재와 규격을 구체적으로 넣었습니다.
기준 컨테이너 4m × 10m 법적 검토 제외 차양 지붕 신규 설치 전제 기존 철골 골조 활용 가정 욕실은 후순위(2차)

핵심 목표

다시 안 새게 만들고, 방/부엌부터 정상화

가장 중요한 게이트

상판·창호 살수 테스트 통과 후 내부 진입

권장 총 공기

1차 3~5주 + 2차 욕실 1주 내외

핵심 순서 요약

이 순서를 바꾸면 재작업 확률이 크게 올라갑니다. 특히 내부 마감 선행이 가장 위험합니다.

습식 전용 공법으로 별도 진행

작업 범위 개념도

이번 문서에서 바꾼 핵심

법적 검토 제외 화장실 후순위 자재 사양 구체화
  • 차양 철거 직후 임시 방수를 별도 단계로 추가
  • 기둥·하부 프레임 점검을 러프인 전 확인 항목으로 승격
  • 벽 덧방 자재를 OSB/MDF → PVC 계열 패널로 변경
  • 상판 방수를 실링 + 방청하도 + 상도 체계로 정리
  • 욕실은 1차 공정에서 분리해 방/부엌 완공 후 진행
기본 가정
실내 높이는 2.3~2.4m 범위로 가정했습니다. 벽 패널 수량은 창호 실제 치수에 따라 최종 조정하세요.

패키지 A · 누수·외피 안정화

게이트: 살수 테스트 통과
A-1 | 1~2일 | 셀프 가능

현장 정리 및 조사

  • 컨테이너 4면 접근 통로 확보
  • 잡목·덩굴·폐기물 정리
  • 전기 차단 구간 정리
  • 누수 포인트, 녹 집중부, 창 하부 상태 사진 기록
A-2 | 1~2일 | 철거 직후 즉시

차양 철거 + 임시 방수

  • 기존 차양 철판·노후 피스 제거
  • 상판 노출 직후 방수포/타프로 임시 보호
  • 비 예보 시 상판 개방 상태로 작업 중단 금지
A-3 | 5~7일 | 숙련자 권장

상판·외벽 보수 + 신규 차양지붕

  • 핀홀/얇아진 철판 패치 또는 용접 보강
  • 기존 실링 제거 후 재실링
  • 녹 제거 → 에폭시 하도 → 우레탄 상도
  • 신규 차양지붕, 후레싱, 홈통 설치
A-4 | 2~3일 | 업체 권장

샷시 교체

  • 경량 이중창 기준
  • 창 상부 물끊기 플래싱 필수
  • 외부 실링 / 내부 마감 분리
  • 교체 후 다시 살수 테스트
패키지 A 통과 기준 — 상판 20~30분 살수 시 실내 누수 없음, 창 상부 누수 없음, 실링 라인 벌어짐 없음.

패키지 B · 방/부엌 우선 정상화

게이트: 누전·누수·배수 테스트 통과
B-1 | 1~2일 | 셀프 가능

내부 선택 철거

  • 천장 합판·마감재 철거
  • 바닥 합판·마감재 철거
  • 싱크대 철거
  • 벽은 상태 양호한 합판 최대한 유지
B-2 | 3~5일 | 숙련자 권장

구조·설비 러프인

  • 하부 프레임·기둥 녹 점검 및 국부 보강
  • 천장/바닥 단열 보강
  • 에어컨 전용 회로, 방수 콘센트 정리
  • 싱크 급배수 재정리, 환기팬 반영
B-3 | 4~6일 | 셀프 + 일부 외주

방/부엌 마감

  • 천장판 재시공
  • 바닥 구조용 합판 복원 + 최종 마감재
  • 벽 몰딩·쫄대 제거 후 얇은 방수 패널 덧방
  • 주방 가구 및 상·하부장 설치
B-4 | 2~3일 | 기사/설치기사 권장

기기 설치 및 최종 점검

  • 싱크 수전·배수 연결
  • 조명·스위치·콘센트 마감
  • 에어컨 신규 설치(최종)
  • 방/부엌 3일 시운전
패키지 B 통과 기준 — 침실 천장 재누수 없음, 바닥 꺼짐 없음, 싱크 배수 정상, 조명·에어컨·콘센트 부하 테스트 이상 없음.

패키지 C · 화장실 2차 공사

방/부엌 안정화 후 별도 진행
C-1 | 1일

전철거 및 바탕 점검

  • 기존 타일·벽체·기구 철거
  • 수손, 곰팡이, 하부 누수 재확인
C-2 | 4~6일 | 숙련자 권장

습식 전용 공법

패키지 C 통과 기준 — 샤워·세면수 사용 후 외부 누수 없음, 코너 실리콘 라인 정상, 변기·세면대 흔들림 없음.

상판 보수 전

  • 핀홀, 용접선 균열, 관통부 위치 표시
  • 고압수는 세척용으로만 사용
  • 이음부·패스너 직분사 금지

외벽 도장 전

  • 뜬 녹과 들뜬 도막 완전 제거
  • 코너·하부 프레임은 패치 여부 먼저 판단
  • 에폭시 하도 후 상도

벽 덧방 전

  • 합판이 물러지지 않는지 눌러 확인
  • 곰팡이 냄새/박리층 있으면 부분 철거
  • 들뜬 페인트 제거 후 덧방

바닥 복원 전

  • 하부 프레임 녹과 단차 확인
  • XPS 단열재 빈틈 없이 충전
  • 구조용 합판은 이음부 엇갈리게 시공

싱크 설치 전

  • 급수·배수 위치 고정
  • 벽체 마감과 콘센트 높이 확정
  • 환기팬 위치 간섭 확인

에어컨 설치 전

  • 벽·천장 최종 마감 완료
  • 배관 관통부와 실외기 위치 확정
  • 배수호스 경사 확보

최종 실행 요약

현장용 한 페이지 버전
1차 공사 목표: 외부가 다시 안 새고, 방/부엌이 정상적으로 쓸 수 있는 상태까지 완성.
실행 순서: 정리 → 차양 철거+임시 방수 → 상판/외벽 보수 → 새 차양지붕+홈통 → 샷시 → 내부 선택 철거 → 구조·설비 러프인 → 천장/바닥/벽 → 싱크/에어컨 → 시운전.
2차 공사 목표: 욕실을 습식 전용 공법으로 별도 재시공.
본 문서는 실행용 설계안입니다. 실제 발주 전에는 창호 치수, 실내 높이, 배수 위치, 기둥/하부 프레임 상태를 현장에서 다시 실측해 수량을 보정하세요.
